## Stack

- Svelte
- CSS
- Vite + Vitest

## Run locally

Create an account [LocationIQ](https://locationiq.com/) to get an access token and add to your `.env` file like this:

```
LOCATION_IQ_ACCESS_TOKEN=token_from_location_iq
```

Then:

```
npm ci
```

And finally:

```
npm run dev
```

You should be ready to go :)

## Tests

Run tests with:

```
npm test
```
